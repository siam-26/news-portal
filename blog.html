  1. Ques:What are the differences between ver,let and const variable?
    Ans:var variables can be updated and re-declared within its scope; let variables can be updated but not re-declared; const variables can neither be updated nor re-declared.


  2. Ques:What are the differences between arrow function and regular function?
    Ans:Regular functions created using function declarations or expressions are constructible and callable, Whereas the arrow functions are only callable and not constructible.


  3. Ques:What are the differences between map,forEach,filter & find?
    Ans:The main differences between them are,forEach() method just loop over the array and executes the callback but not return anything.filter() method executes the callback and check its return value.The map() method creates an entirely new array & returns the newly created array according to the provided callback function.The find() method is used to find all the descendant elements of the selected element.
